What it is:
A simple calculator you can run in Google Colab. You can do basic math like add, subtract, multiply, and divide by typing numbers and choosing an operation. It’s beginner-friendly and runs entirely in your browser—no installation needed.


What it can do:
Add, subtract, multiply, and divide numbers
Works with decimals
Handles errors like dividing by zero
Keeps running until you decide to quit


How to use it:
Open the Colab notebook.
Run the cell with the calculator code.
Follow the prompts to type your numbers and choose an operation.
Type 'q' anytime to stop.


Why it’s useful:
Great for beginners learning Python
No setup required—just a browser and Google account
Helps understand how functions and loops work in Python
